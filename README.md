**Project Overview**
This project is an end-to-end Retail Sales Analytics solution built using SQL Server and Power BI. 
It focuses on cleaning raw sales data, performing exploratory data analysis, calculating key business KPIs, and building an interactive dashboard to support data-driven decision-making.

**Objectives** 
1. Clean and transform raw retail sales data
2. Identify sales trends and business patterns
3. Analyze product, category, and regional performance
4. Build KPIs for business reporting
5. Develop an interactive Power BI dashboard

**Key Analysis Performed** 
1. Monthly and Yearly sales trends
2. Top-performing products
3. Category-wise sales distribution
4. Region-wise performance analysis
5. Running total sales analysis
6. Product ranking using window functions

**Power BI Dashboard Features**
1. KPI Cards (Sales, Profit, Orders)
2. Sales Trend Analysis (Monthly/Yearly)
3. Product Performance Dashboard
4. Regional Sales Map
5. Running Total Visualization
6. Interactive slicers (Category, Region, Date)

**Key Insights**
Identified top-performing product categories driving revenue
Highlighted high-performing regions contributing maximum sales
Observed seasonal sales trends across months
Analyzed product contribution using ranking logic

**Conclusion**
This project demonstrates how raw transactional data can be transformed into meaningful business insights using SQL Server and Power BI, enabling better decision-making for retail business performance optimization.
